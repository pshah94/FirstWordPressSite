Display The Current Date In A Customizable Format

== Description ==

<p>This plugin allows a user to display the current date to the user via a shortcode in any place that a shortcode is usable.</p>
<p>Styling of the date will be the result of styling the shortcode itself. For example if the shortcode is written in bold, blue text,
then the date will show up as bold, blue text</p>
<p>The formatting of the date is accomplished by changing the date format in the WordPress Admin Dashboard under Settings->General. Future
releases of this plugin will allow formatting via the shortcode.</p>
<p>[date today]</p>

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [date today] anywhere that shortcodes are usable.
